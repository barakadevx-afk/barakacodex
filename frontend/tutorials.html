HTML Introduction

Learn the basics of HTML - the foundation of web development

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of web pages.

HTML Example 
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

HTML Structure

Every HTML document follows a basic structure:

  • <!DOCTYPE html> - Document declaration
  • <html> - Root element
  • <head> - Metadata container
  • <body> - Visible content

Key Points

HTML uses "tags" to mark up content. Most tags come in pairs: opening <tag> and closing </tag>.

Always close your tags! Unclosed tags can cause unexpected layout issues.
